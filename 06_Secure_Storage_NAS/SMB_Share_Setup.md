# 📂 SMB Share Setup

## 📌 Overview

An SMB (Server Message Block) share was created to allow Windows systems to access shared storage.

---

## ⚙️ Configuration Steps

* Created dataset: BlueCoast_Shared
* Configured SMB share
* Enabled SMB service

---

## 🎯 Purpose

SMB allows:

* File sharing between systems
* Centralized access for employees
* Integration with Windows environment

---

## ✅ Outcome

Shared folder is accessible across the network.
