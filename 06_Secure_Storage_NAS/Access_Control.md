# 🔐 Access Control

## 📌 Overview

User-based access control was implemented to secure shared storage.

---

## ⚙️ Configuration Steps

* Created user account: blueuser
* Assigned ownership to dataset
* Configured read/write permissions

---

## 🎯 Purpose

Access control ensures:

* Only authorized users access data
* Sensitive files remain protected
* Proper user accountability

---

## ✅ Outcome

Secure access to shared storage is enforced.
