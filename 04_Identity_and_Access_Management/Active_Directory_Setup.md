# 🏢 Active Directory Setup

## 📌 Overview

Windows Server 2025 was configured as a Domain Controller using Active Directory Domain Services (AD DS) to manage users, authentication, and resources centrally.

---

## ⚙️ Installation Steps

1. Opened Server Manager
2. Added "Active Directory Domain Services" role
3. Installed required features
4. Promoted server to Domain Controller

---

## 🌐 Domain Configuration

* Domain Name: **bluecoast.local**
* Forest: New Forest created

---

## 🔐 Purpose

Active Directory enables:

* Centralized authentication
* User and device management
* Policy enforcement across the network

---

## ✅ Outcome

The Windows Server was successfully configured as a Domain Controller and is now managing the enterprise environment.
