# 🔗 Domain Join

## 📌 Overview

The Windows 10 client machine was joined to the Active Directory domain.

---

## ⚙️ Steps

* Accessed system settings
* Joined domain: bluecoast.local
* Authenticated using domain credentials

---

## 🎯 Purpose

Joining the domain enables:

* Centralized login
* Policy enforcement
* Secure access to resources

---

## ✅ Outcome

Windows 10 successfully joined the domain and can authenticate users.
