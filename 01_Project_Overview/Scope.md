# 📦 Project Scope

## ✅ In Scope

The following components are included in this project:

### 🖥️ Infrastructure Setup

* Virtual lab using VirtualBox
* Windows Server (Active Directory, DNS)
* Ubuntu Server (Web hosting)
* Windows 10 client machine
* Kali Linux for attack simulation
* TrueNAS for storage
* Wazuh SIEM for monitoring

---

### 🔐 Security Implementations

* Firewall configuration (Windows & Linux)
* Intrusion Detection System (Snort)
* Brute-force protection (Fail2Ban)
* File integrity monitoring (AIDE)
* Antivirus protection (ClamAV, Windows Defender)
* Multi-Factor Authentication (MFA)

---

### 🌐 Network & Access

* Secure remote access using OpenVPN
* Internal communication via Host-only network
* Controlled access to shared storage

---

### 📊 Monitoring & Logging

* SIEM deployment (Wazuh)
* Centralized logging using Rsyslog
* Log management with Logrotate

---

### 🧪 Testing & Validation

* Brute-force attack simulation using Hydra
* Network traffic analysis using Wireshark
* Vulnerability scanning using OpenVAS

---

## ❌ Out of Scope

The following are not included:

* Deployment to a real production environment
* Cloud infrastructure (AWS, Azure, GCP)
* Advanced enterprise tools (Splunk, Palo Alto, etc.)
* Real customer data handling

---

## ⚠️ Assumptions

* The environment is simulated using virtual machines
* All attacks are conducted in a controlled lab environment
* No real systems are impacted

---

## ✅ Summary

This project focuses on building a secure, simulated enterprise infrastructure to demonstrate practical cybersecurity skills, threat detection, and system hardening.
